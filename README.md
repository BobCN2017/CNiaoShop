# CNiaoShop
在菜鸟窝学习到的菜鸟商场项目，代码全部手敲，并完成课程中未完善的内容，同时对课程中相对难的部分，如购物车的实现，进行了升级。

## 导入时注意事项：
该部分代码为Module部分，无Project内容，如需导入运行，需先新建Project，后使用new->import module方可运行。
